# python-beyond-basics

# Intermediate Python Concepts: Error Handling, File Ops & OOP

Welcome to the second stage of my Python development journey! Having mastered the core fundamentals, this repository focuses on real-world programming concepts: building resilient applications that handle errors gracefully, persisting data using file operations, and writing clean, modular code with Object-Oriented Programming (OOP).

## 🚀 What's Inside?

The repository is structured into intermediate modules, mapping out modern software development workflows:

* **Week 1 - Error Handling (try - except):** Building crash-proof scripts, validation loops, and managing specific exceptions like `ZeroDivisionError` and `IndexError`.
* **Week 2 - File Operations:** Reading and writing text files, dynamic data persistence, and developing a persistent CLI personal journal application.
* **Week 3 - Modules and Libraries:** Exploring built-in utilities like `random` and `datetime`, alongside creating custom local modules for utility functions.
* **Week 4-5 - Introduction to OOP:** Designing modular systems using Classes and Objects, implementing encapsulation, custom exception raising, and class inheritance.
* **Week 6 - Capstone Project:** A mini Library Management System that integrates file persistence, OOP principles, and robust error handling into a single application.
